# Cartomancer

A Tarot-based calendar and divination app for Android, built in Java.

## Features

- Daily Card based on current date
- Personality, Soul, Creativity & Growth Cards calculated from birth date
- Random Card draw with swipe gestures
- Card sorting mini-game (Major Arcana)
- Moon phase display on start screen
- Dark/Light mode toggle
- Navigation drawer with multiple spreads sections

## Status

Work in progress — prototype stage. Future versions planned to include
Maya Tzolkin calendar integration and full astrological overlay.
